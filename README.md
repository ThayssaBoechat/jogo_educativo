# Jogo Educativo de Inglês para Crianças

Um jogo interativo para ajudar crianças a aprenderem inglês de forma divertida.

## Como Jogar

1. Abra o arquivo `index.html` em um navegador moderno
2. Clique nos objetos na tela
3. Ouça a pronúncia em português e inglês

## Estrutura do Projeto

- `index.html` - Página principal do jogo
- `styles.css` - Estilos e layout do jogo
- `script.js` - Lógica e interatividade do jogo
- `images/` - Pasta contendo todas as imagens do jogo

## Requisitos

- Navegador moderno com suporte a:
  - HTML5
  - CSS3
  - JavaScript ES6+
  - Web Speech API
